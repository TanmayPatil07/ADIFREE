# ADIFREE
Adi Free is a web platform that helps tobacco and cigarette users follow a structured 21-day de-addiction program. It features secure login/signup, day-wise locked progress, weekly difficulty progression, customized plans via email request, and nearby rehabilitation hospital contacts for professional support and physical checkups.
